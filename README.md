# Planner & Calendar App

A React + TypeScript + Vite application for weekly planning, task management, and yearly calendar overview.

## Features

- **Weekly View**: detailed task management with drag-and-drop-like organization (groups, days).
- **Calendar View**: Yearly, Monthly, Weekly, and Daily overviews of tasks and events.
- **Markdown-First**: Data model designed around Markdown serialization.
- **Dark Mode**: sleek dark UI using Tailwind CSS.

## Getting Started

1. `npm install`
2. `npm run dev`

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- React Big Calendar
- Date-fns
